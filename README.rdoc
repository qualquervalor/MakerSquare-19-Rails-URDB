MakerSquare_19 Week 5 URDB
This is an assignment from the 5th week of MakerSquare 10 week course on web development. The project is a Rails 
clone of IMDB, which would list movie and have detailed information available for each movie.
Each day work would begin anew on a fresh branch to teach different concepts.
-Generating rails migration
-Adding model associations (has_many :through)
-Using Haml for Views
-Testig with RSpec
